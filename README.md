# 4JU

기존 사주 리포트 사이트 배포본.

- 기준 원본: 사용자가 제공한 Cloudflare live preview
- GitHub Pages 배포: `gh-pages` branch
- 현재는 기존 생년월일/출생시간/이메일/무료 요약 플로우를 그대로 보존
- 프리미엄 기능
  - 프리미엄 상세 바텀시트
  - 4,900원 상품 구성 카드
  - 돈/일, 관계, 30일 루틴 샘플 전환
  - 결제 연결 준비 CTA
  - 프리미엄 미리보기 블러 카드
- 테스트 모드
  - 기본값은 프리미엄 열린 상태
  - `?premium=0`을 붙이면 무료/잠금 상태 테스트 가능
  - 실제 결제 없이 `프리미엄 열기` 버튼으로도 결제 완료 상태 확인 가능
- 계산 UX
  - 결과 표시 전 2.4초 계산중 오버레이
  - 단계: 생년월일 확인 → 일간·오행 계산 → 무료 리포트 구성
  - 스피너 + 진행바 애니메이션

> 전통 사주명리학의 상징 체계를 바탕으로 만든 자기이해/엔터테인먼트용 리포트입니다. 의료·법률·투자·진로 결정을 대신하지 않습니다.
