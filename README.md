# 4JU

Toss-blue 스타일의 생활형 사주 리포트 MVP.

## 현재 기능

- 생년월일 8자리 입력 플로우
- 출생시간 모름/시간 선택 플로우
- 무료 기본 사주 리포트
- 실제 간지 계산 기반 년주·월주·일주·시주 표시
- 계산값 → 명리 기본 → 현대식 번역 근거 체인 표시
- 일간별 기본 설명/핵심 성향/일하는 방식/주의점/루틴 3개
- 오행 흐름 미리보기
- 가족 궁합 기능
  - 엄마/아빠/아이 입력
  - 부모-아이 궁합 점수
  - 일간 오행 생극 + 일지 합/충/해 기반 간단 설명
  - 예시 가족 불러오기

> 전통 사주명리학의 상징 체계를 바탕으로 만든 자기이해/엔터테인먼트용 리포트입니다. 의료·법률·투자·진로 결정을 대신하지 않습니다.
