# 4JU

기존 사주 리포트 사이트 배포본.

- 기준 원본: 사용자가 제공한 Cloudflare live preview
- GitHub Pages 배포: `gh-pages` branch
- 현재는 기존 생년월일/출생시간/이메일/무료 요약 플로우를 그대로 보존
- 프리미엄 기능
  - 기본값: 테스트 결제 완료 상태로 프리미엄 리포트 자동 오픈
  - 무료/잠금 상태 테스트: `?premium=0`
  - 상세 리포트 강화
    - 돈/일 스타일: 근거 체인, 잘 맞는 일, 피해야 할 방식, 수익화 팁
    - 관계 패턴: 반복 갈등, 좋은 말투, 회복 루틴
    - 올해 흐름: 상반기/하반기/주의 포인트
    - 30일 실행 처방: 1~4주차 루틴
    - 가족/궁합: 생활 운영법 중심
  - PDF 저장 / 공유 이미지 버튼 자리
- 계산 UX
  - 무료 결과 표시 전 계산중 오버레이
  - 단계: 생년월일 확인 → 일간·오행 계산 → 실시간 AI 해석 → 결과 정리
  - 스피너 + 진행바 애니메이션
- AI 리포트
  - Pollinations 무료 텍스트 API 연결
  - 결과 화면에 `AI 호출중` / `AI 호출 성공` / `기본해석 표시` 배지 표시
  - 호출 성공 시: Pollinations LLM 응답으로 요약/프리미엄 문장 교체
  - 프롬프트에서 AI 말투 금지: 핵심은/중요한 건/정리하면/구조화/활용/가능합니다 등 제외
  - 응답 후처리로 딱딱한 AI식 표현을 한 번 더 제거
  - 실패/지연/큐풀/오류 시: 템플릿 기반 기본 해석 표시 + 실패 사유 노출

> 전통 사주명리학의 상징 체계를 바탕으로 만든 자기이해/엔터테인먼트용 리포트입니다. 의료·법률·투자·진로 결정을 대신하지 않습니다.
